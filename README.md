# Spotify Premium Mod v4.2.1

Unlocked Spotify Premium client for Windows. No ads, unlimited skips, high quality streaming, and offline downloads.

## Features
- No advertisements (audio or visual)
- Unlimited skips
- Very High quality streaming (320kbps)
- Offline download support
- Spotify Connect enabled
- No forced shuffle on albums

## System Requirements
- Windows 10/11 64-bit
- Python 3.11+
- Spotify desktop app installed (free version)

## Quick Start
1. Extract all files to a folder
2. Run `install.bat` as Administrator
3. Launch Spotify normally
4. The mod auto-injects on startup

## How It Works
This mod patches the Spotify desktop client in memory to enable Premium features. It does not modify any Spotify files on disk.

## Troubleshooting
- "Injection failed": Make sure Spotify is closed before running install.bat
- "Ads still playing": Restart Spotify after installation
- Antivirus alert: Add folder exclusion (false positive on memory patching)
